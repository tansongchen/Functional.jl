函数式编程

写好函数，而不是写好脚本

函数式编程意味着将程序运行的结果表达为输入数据的函数。在实际运用中，它表现出了以下特点：

  1. 函数的返回值完全由参数确定；
  2. 函数不改变参数；
  3. 除非出于输入/输出和内存管理的需要，尽量使得对象不改变。
